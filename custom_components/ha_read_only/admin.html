Anleitung

So nutzt du die HA Read-Only API

Was ist das?

Die HA Read-Only API bietet einen sicheren, schreibgeschützten Zugriff auf deine Home-Assistant-Entitäten über einfache REST-Endpunkte. Ideal für Wandtablets, externe Dashboards oder Monitoring-Tools, die deine Daten nur lesen – aber niemals ändern – sollen.

1. Token erstellen

a) Klicke oben auf + Neuer Token

b) Vergib einen aussagekräftigen Namen (z.B. "Wand-Tablet Küche")

c) Wähle die erlaubten Domains aus (z.B. nur sensor und light)

d) Optional: Setze Whitelist/Blacklist-Muster für feingranulare Kontrolle

e) Kopiere den generierten Token sofort – er wird nur einmal angezeigt!

2. API-Endpunkte

Alle Anfragen benötigen den Header X-HA-ReadOnly-Token: DEIN_TOKEN

Alle Zustände abrufen

GET /api/ha_read_only/states

Gibt alle erlaubten Entitäten mit Status und Attributen zurück.

Einzelne Entität abrufen

GET /api/ha_read_only/states/{entity_id}

z.B. /api/ha_read_only/states/sensor.temperatur_wohnzimmer

Entitätenliste abrufen

GET /api/ha_read_only/entities

Gibt nur die Entity-IDs zurück (ohne Zustände/Attribute).

3. Beispiele

cURL:

curl -H "X-HA-ReadOnly-Token: DEIN_TOKEN" \
  http://DEINE_HA_IP:8123/api/ha_read_only/states

JavaScript (fetch):

const res = await fetch('/api/ha_read_only/states', {
  headers: { 'X-HA-ReadOnly-Token': 'DEIN_TOKEN' }
});
const daten = await res.json();

Python:

import requests
r = requests.get(
  'http://DEINE_HA_IP:8123/api/ha_read_only/states',
  headers={'X-HA-ReadOnly-Token': 'DEIN_TOKEN'}
)
print(r.json())

4. Filter-Logik

Domains: Wähle ganze Kategorien (z.B. alle Sensoren, alle Lichter)

Erlaubte Muster: Glob-Muster wie sensor.*temperatur* für feingranulare Whitelist

Blockierte Muster: Glob-Muster wie *passwort* zum Ausschließen (hat Vorrang!)

Keine Filter? Dann sind automatisch alle Entitäten erlaubt

5. Sicherheitshinweise

Tokens sind schreibgeschützt – niemand kann über diese API Geräte steuern

Erstelle separate Tokens pro Gerät/Anwendung

Bei Verdacht auf Missbrauch: Token neu generieren oder löschen

Nutze die Rate-Limits unter Einstellungen zum Schutz vor Überlastung
